# 🎉 项目简介

---

## 1. 总体目标 

按照图书名称进行笔记查找。即允许用户上传某本书的笔记，其他用户可以根据该本书查找到所有用户的笔记内容（可设置为私有）

## 3. 技术栈

**前端技术栈：**

**后端技术栈：**

- Spring Boot

- Apache Log4j2

- Spring Data Redis

- MySQL
- Redis
